SocialMini - Social Media Platform

A lightweight, full-stack social media application inspired by Facebook. Built with PHP (Native), MySQL, and styled with Tailwind CSS.

This project features a robust authentication system, real-time-simulated chat, interactive posts, and a modern Dark Mode UI.

âš™ï¸ Installation Guide

Prerequisites

XAMPP (or any Apache/MySQL server environment).

A web browser.

Step-by-Step Setup

Clone/Download:

Navigate to your server's root directory (e.g., C:\xampp\htdocs).

Create a folder named social_app.

Place all project files inside this folder.

Database Configuration:

Open phpMyAdmin (http://localhost/phpmyadmin).

Create a new database named social_db.

Import the following SQL Schema:

-- Users Table
CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    first_name VARCHAR(50),
    last_name VARCHAR(50),
    email VARCHAR(100) UNIQUE,
    password VARCHAR(255),
    avatar VARCHAR(255) DEFAULT '[https://api.dicebear.com/7.x/avataaars/svg?seed=Default](https://api.dicebear.com/7.x/avataaars/svg?seed=Default)',
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- Posts Table
CREATE TABLE posts (
    id INT AUTO_INCREMENT PRIMARY KEY,
    user_id INT,
    content TEXT,
    image VARCHAR(255),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id)
);

-- Likes, Comments, Friends, Messages, Notifications
-- (See implementation guide for full schema)


Configure Connection:

Ensure includes/db_connect.php matches your local MySQL credentials (default: root, password: empty).

Run:

Start Apache and MySQL in XAMPP control panel.

Open your browser and visit: http://localhost/social_app/login.php
